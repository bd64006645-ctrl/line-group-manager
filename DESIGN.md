# DESIGN.md

## 项目与用户画像
- LINE 群管理工具后台，面向运营代理人和超级管理员
- 用户场景：日常群组监控、敏感词配置、禁言管理、事件日志查看
- 使用频率高，需要快速定位和操作

## 气质与意象
- **关键词**：清晰、可控、专业、高效
- **意象**：清晨的运营指挥中心——大面积留白、信息层次分明、一眼看到关键数据，不花哨但每个元素都有功能

## 配色方案
- **主色**：LINE Green `#06C755` — 品牌识别色，用于主按钮、激活态、状态指示
- **辅色**：深灰 `#1A1A2E` — 侧边栏背景，沉稳专业
- **背景**：浅灰 `#F8F9FA` — 主内容区背景，减少视觉疲劳
- **卡片**：纯白 `#FFFFFF` — 信息容器，干净通透
- **文字**：深灰 `#2D3436` 主文本 / `#636E72` 次要文本
- **状态色**：成功绿 `#00B894` / 警告橙 `#FDCB6E` / 危险红 `#E17055` / 信息蓝 `#74B9FF`

## 字体排版
- **字体族**：PingFang SC, Hiragino Sans GB, Microsoft YaHei, system-ui
- **标题**：18-24px, font-weight 600
- **正文**：14px, font-weight 400
- **辅助**：12px, font-weight 400, color muted

## 布局与响应式
- 左侧固定侧边栏（240px），顶部面包屑/搜索栏
- 主内容区卡片式布局，grid 自适应
- 移动端侧边栏折叠为抽屉

## 组件规范
- 基于 shadcn/ui 组件库
- 卡片统一圆角 `rounded-xl`，轻阴影 `shadow-sm`
- 表格使用紧凑模式，行高适中
- 状态标签使用 Badge 组件，颜色对应状态色

## 动效与交互
- 页面切换：fade-in 150ms
- 卡片悬停：微上移 `translateY(-1px)` + 阴影加深
- 按钮点击：scale 0.98 反馈
- 开关切换：平滑过渡 200ms

## 设计禁忌
- 不要使用深色/暗黑主题
- 不要使用渐变背景或大面积彩色填充
- 不要使用过度圆角（>16px）或过度阴影
- 不要使用动画图标或闪烁效果
